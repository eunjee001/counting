# 숫자 세기(계수기) 앱

* linearlayout을 이용하여 화면 그림
* 유저 인터페이스 구성

## 개요
* 버튼 누르면 숫자가 1씩 올라감
* 초기화 기능 있음
* 9999까지 숫자 셀수 있음

## 구현 기능
* '+' 버튼 클릭시, 숫자 1씩 올리기
* 초기화 버튼 클릭시, 숫자 0으로 변경

UI
* Linealayout
* TextView
* Button

Kotlin
